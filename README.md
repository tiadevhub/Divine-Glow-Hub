# ✨ Divine Glow Hub

A luxury skincare website developed using HTML, CSS, JavaScript, API integration and Local Storage.

## 📌 Project Overview

Divine Glow Hub is a skincare shopping website where users can explore skincare collections, create custom kits, place orders, track deliveries and manage their shopping experience.

This project was created as part of a Front-End Web Development evaluation.

---

## 🚀 Features

### 🏠 Home Page
- Responsive landing page
- Modern skincare theme

### 🧴 Product Section
- Browse skincare products
- Create custom skincare kits
- Dynamic pricing updates

### 🛒 Cart System
- Add to cart
- Store cart using Local Storage

### 👤 Authentication
- Login
- Signup
- Session handling using Local Storage

### ⭐ Review System
- Submit reviews
- Rating feature
- Delete own reviews only

### 📦 Order Management
- Place order
- Order tracking
- Delivery status updates
- Cancel order before shipping

### 🔎 Search Functionality
- Product search
- No result message

### 🌐 API Integration
- Daily quote API
